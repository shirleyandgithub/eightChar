# eightChar
国学命理app-服务端代码

本项目为八字排盘工具 ，可根据已知时间、四柱进行排盘！ 排盘界面分命主信息、基本命盘、顺物。 排盘数据包含四柱、十神、藏干、纳音、星运、空亡、神煞、天罡称骨、五行数据等信息。

该项目仅为服务端代码， 技术栈： JAVA + SpringBoot + MyBatis + Mysql + Redis

对应的前端代码请看这里：https://github.com/shirleyandgithub/eightChar-web
